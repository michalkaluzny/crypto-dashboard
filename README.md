# Crypto Dashboard - web application to live track crypto price and news 

# Functions
- Track bitcoin price in real time
- Check changes in bitcoin price on chart
- Check latest news about bitcoin
- Ask ai chatbot about cryptocurrencies

# Tech stack
- Backend: Python
- Frontend: React

# How to run
First terminal:
1. change the api_key in chatboy.py to your openai api key
2. git clone https://github.com/michalkaluzny/crypto-dashboard
3. cd crypto-dashboard/backend
4. pip install -r requirements.txt
5. uvicorn main:app --reload
Second terminal:
6. cd frontend
7. npm install 
8. npm run dev 
9. enter the localhost
